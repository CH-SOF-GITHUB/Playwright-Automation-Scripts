# Playwright Automation Scripts

Ce projet contient des scripts d'automatisation de tests utilisant Playwright pour vérifier différentes fonctionnalités d'un site e-commerce.

## Fonctionnalités
- Connexion avec identifiants
- Vérification du chargement des pages
- Navigation et modification du profil utilisateur
- Déconnexion

## Technologies
- Playwright
- TypeScript

## Comment utiliser
1. Cloner le dépôt
2. Installer les dépendances : `npm install`
3. Lancer les tests : `npx playwright test`
